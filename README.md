# Personal Portfolio Website

## Project Description
This is my personal portfolio website created using HTML, CSS, and JavaScript.  
It showcases my projects, skills, and background as a student learning web development.

The website includes multiple pages:
- Home
- About Me
- Projects
- Skills
- Contact

It also includes interactive features such as a JavaScript button interaction and responsive design.

---

## How to View the Website Locally

To run this project on your computer:

1. Download or clone the repository:
2. Open the project folder.

3. Open `index.html` in any web browser (Chrome, Edge, etc.)

No additional setup is required.
